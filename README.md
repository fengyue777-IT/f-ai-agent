## 📖 项目概述

**车载AI亲人声音情感助手** 是一个专为独自驾驶者设计的AI原生应用。它通过克隆亲人声音，在行程中提供**自然的情感陪伴**，并在突发紧急情况时，触发集**精准呼救与情感安抚**于一体的安全保障。我们相信，因为有了AI，驾驶舱可以不再是情感孤岛，也能成为最可靠的安全港湾。

> **核心价值**：用“亲人声音”这一最强情感锚点，同时解决驾驶中的 **“情感孤单”** 与 **“遇险无助”** 两大核心痛点。

## ✨ 核心特性与创新点

### 1. 🧠 **AI 原生的情感体验**
- **深度情感连接**：不是冰冷的机器语音，而是用户熟悉的“妈妈”、“爸爸”的声音在关心和安抚。
- **场景化智能交互**：覆盖“通勤闲聊”、“长途解闷”、“情绪倾诉”场景，AI主动发起话题，无需用户费心寻找。

### 2. 🚨 **首创：情感化紧急呼救（核心差异化）**
- **一键语音触发**：遇险时只需喊“**妈，救命**”，系统即启动全方位救援。
- **自动化精准求助**：自动拨打120/119/110，并**同步实时位置**给预设紧急联系人。
- **恐慌情绪安抚**：在整个呼救过程中，持续用“亲人声音”进行语言安抚，缓解用户恐慌，这是普通紧急呼叫完全不具备的**人性化关怀**。

### 3. ⚡ **极致的车载场景适配**
- **零学习成本**：全程语音交互，免唤醒词，用日常称呼（如“爸”、“妈”）直接发起指令。
- **1分钟快速上手**：上传3-5段语音即可克隆音色，紧急联系人设置一步完成。
- **驾驶安全优先**：交互响应简洁，并具备防误触设计和疲劳驾驶提醒。

## 🛠️ 技术栈与开发说明

**本项目的全部代码均在 AI 编程助手（Cursor、GitHub Copilot）的辅助下完成，大幅提升了开发效率与代码质量。**

| 模块 | 技术/服务选择 | 说明 |
| :--- | :--- | :--- |
| **前端 (客户端)** | **Flutter (Web)** | 一套代码可编译为Web应用，方便演示，未来可轻松扩展至Android/iOS车机。 |
| **后端/Agent服务** | **Python (FastAPI)** | 轻量、异步，快速构建API。部署于Vercel/Replit等Serverless平台。 |
| **核心AI能力** | **1. 语音克隆**: 讯飞开放平台API <br> **2. 对话与情绪理解**: DeepSeek/OpenAI API <br> **3. 语音合成(TTS)**: 克隆音色 + 情绪化语调生成 | 均采用成熟API，无需训练模型，聚焦应用集成。 |
| **紧急呼救实现** | **1. 拨号**: 使用`tel:`协议唤起系统电话 <br> **2. 定位**: 浏览器Geolocation API + 高德地图逆地理编码 <br> **3. 通知**: 调用短信/邮件接口（如腾讯云短信）或展示分享面板 | 调用系统级能力与轻量级云服务，5天内可完成集成。 |
| **开发工具** | **Cursor, GitHub Copilot** | 用于代码生成、解释、重构及文档撰写。 |

### 启动应用
1.  **启动后端服务**
    ```bash
    cd backend
    uvicorn main:app --reload
    ```
    服务将在 `http://localhost:8000` 启动。
2.  **启动前端应用**
    ```bash
    cd frontend
    flutter run -d chrome
    ```
    应用将在Chrome浏览器中打开，默认连接到本地后端。

### 核心功能演示流程
1.  **首次使用**：进入应用，跟随语音引导，在1分钟内完成“上传亲人语音”和“设置紧急联系人”。
2.  **日常陪伴**：
    - 说：“**妈，今天有点累。**”
    - AI（用妈妈声音）回应：“累坏了吧，专心开车，回家给你炖汤。”
3.  **紧急呼救**：
    - 遇险时大喊：“**妈，救命！**”
    - AI迅速确认：“是不是遇到紧急情况？需要马上呼救吗？”
    - 用户回应：“**是！**”
    - **系统自动执行**：①拨打120 ②将实时位置短信发送给家人 ③用妈妈声音循环安抚：“别怕，救护车已经在路上，我在这儿陪着你。”
